
# Maude Notepad++
Language Definition for Maude in Notepad++.

## Installation

 - Clone XML file
 - Click on Language (on the Menu Bar)
 - Define your language...
 - Import...
 - Select the cloned XML
 - Close Notepad++

## Usage

 - Click on Language (on the Menu Bar)
 - Select the language (usually it is on the bottom)
